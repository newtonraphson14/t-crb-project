# Interim Images

Folder ini untuk hasil pemrosesan image sebelum jadi artefak final.

Subfolder yang disarankan:

- `aligned/` untuk cutout yang sudah dipusatkan ke T CrB
- `reprojected/` untuk image yang sudah dibawa ke WCS target sama
- `normalized/` untuk PNG/FITS yang sudah di-stretch untuk display
- `masks/` untuk footprint, bad-pixel mask, atau mask lain

Lihat `notes/raw_image_pipeline_map.md` untuk naming convention dan relasinya ke lane `modern V` dan `all-cycles Vis`.
